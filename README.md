# dhakamythos

Site for the dhakamythos art collective. SvelteKit (Svelte 5) · Tailwind +
CSS tokens · GSAP + native View Transitions · Sanity (embedded `studio/`) ·
Vercel.

Read first: `CLAUDE.md` (rules) · `docs/DESIGN.md` · `docs/SITEMAP.md` ·
`docs/PERFORMANCE.md`. Specs win over defaults.

```
npm run dev          # site
npm run studio       # sanity studio
npm run build && npm run preview
npm run perf         # lighthouse ship gate
```

Publishing how-to for the collective lands here at M5.
